PROGRAMMING LANGUAGES


A Programming language is a formal language designed into communicate instructions to machine,particularly a computer programming language can be used to create programs that control the behaviour of machine and or express algorithm precisely.


C++


C++ is a high-level programming language developed by Bjarne Stroustrup at Bell Labs. C++ adds object-oriented features to its predecessor, C. C++ is one of the most popular programming language for graphical applications, such as those that run in Windows and Macintosh environments.

C


C is a high-level and general purpose programming language that is ideal for developing firmware or portable applications. Originally intended for writing system software, C was developed at Bell Labs by Dennis Ritchie for the Unix Operating System (OS) in the early 1970s.

JAVA


Java is an object-oriented language similar to C++, but simplified to eliminate language features that cause common programming errors.
